A beginner-friendly **Blog Web Application** where users can register, login, and create, edit, or delete their own blog posts. Posts can include images, titles, descriptions, and categories such as **Art, Science, Technology, Cinema, Design, and Food**.  

Built with **React** for the front-end and **Node.js** for the back-end.

- **User Authentication**

- **Categories**
  - Filter posts by **Art, Science, Technology, Cinema, Design, Food**
- **Responsive UI**
